Sistema de Matrícula do Aluno

Use atributos múltiplos para o cadastro, como nome do aluno + curso

O usuário do sistema deve:

  1. Fazer o cadastro das informações do aluno previamente no módulo: Cadastro de Alunos
  2. No sistema de cadastro do curso disponível em Cadastro de Cursos, o usuário do sistema deve cadastrar os cursos oferecidos pela instituição
  3. Ao fazer o cadastro da matrícula, o usuário do sistema deve selecionar o aluno previamente cadastrado e o curso oferecido pela instituição.
  4. Para concluir a matrícula, o usuário do sistema deve clicar no botão "Concluir Matrícula".
